### Launch programs

#### 1.Install NodeJs and npm package manager

#### 2.Install dependencies before you run:
```
npm install
```
(Run this once in 2025 directory)

#### 3. Run the program with this:
```
npx tsx <filename.ts>
```

Or you can use in linux:

```
./runA.sh

./runB.sh
```
